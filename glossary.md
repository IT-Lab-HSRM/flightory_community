# 📖 Glossary

A reference guide for common terms used in fixed-wing and VTOL drone building, FPV flying, 3D printing, and flight electronics.

| Term | Description |
|---|---|
| **Airframe** | The main structure of the aircraft — fuselage, wings, and tail. |
| **Airspeed Sensor** | Measures how fast the plane is moving through the air (not GPS speed). |
| **Autotune** | Automatically adjusts control gains for optimized flight behavior. |
| **Barometer (Baro)** | Measures altitude via air pressure. |
| **BEC (Battery Eliminator Circuit)** | Supplies power to the flight controller and servos from the battery. |
| **Brim / Raft** | Extra printed material added around the base of a print to improve bed adhesion. |
| **CA** | Cyanoacrylate — it's the chemical name for super glue. |
| **CG (Center of Gravity)** | The balance point of the aircraft — crucial for stable and predictable flight. |
| **CRSF (Crossfire)** | Protocol used by the TBS Crossfire system. Very fast and low-latency, with telemetry support and extended features. |
| **Dihedral** | The upward angle of the wings from the center, used to improve passive stability. |
| **Diversity** | A dual-antenna system that automatically selects the stronger signal source. |
| **DVR (Digital Video Recorder)** | Records the FPV video feed — often built directly into goggles. |
| **Elevon** | A control surface that acts as both elevator and aileron. Common in flying wing designs. |
| **ELRS (ExpressLRS)** | Open-source, ultra-fast, long-range RC protocol. Becoming the go-to standard in FPV — low latency, high refresh rate, telemetry included. |
| **ESC (Electronic Speed Controller)** | Controls the motor's speed based on signals from the flight controller. |
| **Failsafe** | A safety mode triggered automatically by signal loss or critical errors. |
| **FC (Flight Controller)** | The onboard computer that manages sensors, stabilization, and automation. |
| **Fixed-Wing** | A type of aircraft with stationary wings, as opposed to multirotor drones. |
| **FPV (First Person View)** | Flying using a live video feed from an onboard camera. |
| **G-code** | Instructions generated by slicer software to control a 3D printer's movements and extrusion. |
| **Goggles** | Head-mounted display used for FPV flying. |
| **GPS** | Provides real-time location, altitude, and heading data. |
| **Gyro / IMU (Inertial Measurement Unit)** | Measures rotation, pitch, and movement — the core of any flight controller. |
| **Infill** | The internal structure of a 3D print, expressed as a percentage and pattern (e.g. 15% honeycomb). |
| **Latency** | Time delay between camera capture and display in an FPV system. Lower is always better. |
| **Layer Height** | The thickness of each individual printed layer (e.g. 0.2 mm). |
| **LW-PLA / LW-ASA** | Lightweight foaming filaments specifically designed for RC aircraft — very low density once printed. |
| **Nozzle** | The tip of the hotend through which filament is extruded during 3D printing. |
| **OSD (On-Screen Display)** | Overlays live flight data (altitude, voltage, speed, etc.) onto the FPV video feed. |
| **PID** | Control loop parameters (Proportional, Integral, Derivative) that determine how the aircraft responds to inputs and disturbances. |
| **Print Bed / Build Plate** | The surface on which 3D objects are printed. |
| **Pusher** | A motor mounted at the rear of the aircraft, pushing it forward. |
| **Retraction** | The pullback of filament during non-print moves to prevent stringing or oozing. |
| **RSSI (Received Signal Strength Indicator)** | Indicates the quality of the RC link signal. |
| **RX / TX** | Receiver (on the aircraft) / Transmitter (in your hands). |
| **Slicer** | Software that converts 3D models into G-code for the printer (e.g. OrcaSlicer, Cura, BambuStudio). |
| **STEP (Standard for the Exchange of Product Data)** | An editable CAD file format — used when you need to customize a model before printing. |
| **STL (Stereolithography)** | A 3D model file format used for slicing and printing. Not directly editable in most CAD tools. |
| **Supports** | Temporary printed structures that support overhanging parts of a model during printing. |
| **Telemetry** | Sends live flight data (altitude, voltage, GPS, etc.) to a ground station in real time. |
| **Tractor (Puller)** | A motor mounted at the front of the aircraft, pulling it forward. |
| **V-tail** | A tail configuration shaped like a "V", combining elevator and rudder control into two surfaces. |
| **VRX (Video Receiver)** | Receives the FPV video signal transmitted from the drone. |
| **VTX (Video Transmitter)** | Sends the FPV video signal from the drone to goggles or a screen. |
| **Warping** | Curling or lifting of a 3D print caused by uneven cooling during the print process. |
| **Wing Loading** | The ratio of aircraft weight to wing area. Lower wing loading means more efficient and stable flight. |
| **Z-hop** | A small upward movement of the nozzle during travel moves to avoid hitting already-printed parts. |
